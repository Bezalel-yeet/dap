# Degree Apprenticeship Preparation

A central hub for tracking progress and gathering useful information in preparation for degree apprenticeship applications.

---

## Purpose

This repo serves as a personal tracker and knowledge base to:

- Stay organised throughout the application process
- Record research, deadlines, and decisions
- Collect useful resources, links, and notes
- Monitor progress across each stage of the journey
